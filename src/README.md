# Лабораторна робота №2: Алгоритми розгалуженої структури

## Мета роботи
Опанування навичок програмування розгалужених процесів за допомогою умовного оператора `if` та оператора вибору `switch` у мові Java.

## Структура проєкту
Проєкт складається з окремих класів для кожного завдання, що викликаються в головному файлі `Main.java`:

- **Main.java** — точка входу, яка по черзі запускає всі завдання.
- **Task1.java** — обчислення складної функції f(x) за умовами варіанту 8.
- **Task2.java** — реалізація 7 підзадач з використанням оператора `switch`.
- **Task3.java** — розв'язання квадратного рівняння.
- **Task4.java** — визначення координатної чверті точки.
- **Task5.java** — логічна перевірка числа (ціле, парне, двозначне).
- **safeEngine.java** - Містить методи `readInt` та `readDouble` з обробкою виключень `InputMismatchException`, що запобігає крашу програми при введенні літер.

## Опис завдань



### 1. Обчислення функції (Варіант 8)
Обчислюється значення f(x) залежно від діапазону:
- x < -2 (f(x) = e^(2x))
- -2 <= x <= 5 f(x) = ctg(x) = 1 / tg(x)
- x > 5 f(x) = x^4

### 2. Оператор switch (7 пунктів)
Реалізовано наступні модулі:
1. **Тип дня**: Визначення, чи є день тижня робочим (1-5) чи вихідним (6-7).
2. **Пори року**: Вивід пори року за номером місяця.
3. **Кількість днів**: Вивід кількості днів у місяці (враховуючи лютий).
4. **Текстове меню**: Вивід повідомлень (привітання/запрошення/завершення).
5. **Розклад**: Вивід назви предмету за номером пари (1-4).
6. **Чверть години**: Визначення чверті години за введеними хвилинами (0-59).
7. **Арифметичне меню**: Вибір операції (сума, різниця, добуток) для двох чисел.

### 3. Квадратне рівняння
Програма приймає коефіцієнти a, b, c та знаходить корені рівняння ax^2 + bx + c = 0, аналізуючи дискримінант ($D > 0, D = 0, D < 0$).

### 4. Координатна чверть
За введеними координатами (x, y) визначається номер квадранта (I, II, III або IV).

### 5. Перевірка числа
Реалізовано алгоритм, який:
1. Перевіряє, чи є число цілим (використовуючи `input % 1 == 0`).
2. Визначає, чи є воно парним.
3. Перевіряє, чи є воно двозначним (в діапазоні [10, 99] або [-99, -10]).
